1. Java と C++ の違いを調べてみましょう

	1. プログラミング言語の種類
	   Javaはオブジェクト指向プログラミング言語
	   　→クラスとオブジェクトの概念に基づいている
	   C++はマルチパラダイム言語
	   　→手続き型プログラミングとオブジェクト指向プログラミングの両方の要素を持つ

	2. メモリ管理:
	   Javaはガベージコレクション（自動メモリ管理）を使用している
	   　→開発者が明示的にメモリを解放する必要がない。
	   C++では、メモリの割り当てと解放を手動で管理する必要がある。

2. Java と JavaScript の違いを調べてみましょう
	Java: 主にバックエンド開発やデスクトップアプリケーションの開発に使用されている
	　　　プラットフォームに依存しない言語であり、さまざまなアプリケーションやシステムで利用される
	　　　→コンパイルすることで、動作環境に依存せず実行することができる。
	JavaScript: JavaScriptは主にウェブ開発で使用されるスクリプト言語
	　　　動的なウェブページを作成するために使用されている。
	　　　→原則ブラウザ上で実行される。
	　　　
3. Java と Python の違いを調べてみましょう
　　Java：静的型付け言語であり、厳密な型宣言が必要となる。
　　　　　変数の型は事前に宣言する必要がある。
　　Python：動的型付け言語であり、型の宣言が不要
　　　　　変数の型は実行時に自動的に定義される。
